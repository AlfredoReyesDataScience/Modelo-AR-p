## Dataset

No hubo un dataset como tal, sino que sacamos los datos de la empresa Walt Disney Company de https://finance.yahoo.com/ 

## Tecnicas utilizadas

Modelo AR(p)

Uso de gráfica de Auto-Correlación Parcial

Estadísticos de Akaike y Bayesiano
